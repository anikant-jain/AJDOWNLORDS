# AJ Downloader 🎬

A slick YouTube & Instagram downloader with a 3D animated UI.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Install [FFmpeg](https://ffmpeg.org/download.html) (needed for MP3 conversion)

3. Run the app:
```bash
python app.py
```

4. Open your browser at `http://127.0.0.1:5000`

## Features
- Download YouTube videos & Shorts
- Download Instagram Reels & Posts
- MP4 video or MP3 audio
- 3D animated background
